En el escritorio aparecen una gran cantidad de iconos, vamos a empezar ejecutando hashtag que contiene unas opciones básicas. También podemos ejecutarlo directamente escribiendo "hashtag" sin las comillas en una terminal.

La primera opción incluye este manual que documenta Hashtag-Linux.

La segunda opción llamada DDB nos permite grabar una iso a un usb, mediante el programa dd.

Buscamos la iso y la seleccionamos, el archivo debe acabar en .iso

Aquí aparece la lista de usb conectados al equipo, sólo soporta hasta sdd, es decir sólo 3 dispositivos aparte de sda.

Para ejecutar dd se necesitan privilegios, así que se necesita autenticación.

Tras finalizar la copia, podemos utilizar el usb para instalar el sistema operativo que contenía la iso.

VIRTUALBOX

Una vez seleccionada una máquina virtual deseada, podemos cambiar la configuración de ésta, también podemos iniciarla o descartar el estado actual de la máquina si no se ha apagado para apagarla sin necesidad de iniciarla. Además podemos crear una nueva si disponemos de un .vdi o de una .iso, existen diferentes formas de iniciar una máquina virtual; en pantalla completa o en modo escalado entre otras opciones.

Permite ajustar la memoria ram dedicada a la máquina virtual, por defecto tiene 768 MB.

Permite ajustar la memoria gráfica dedicada a la memoria virtual, por defecto utiliza 64 MB.

Permite cambiar el adaptador de red o deshabilitarlo, por defecto está conectado a NAT. Permite elegir como la máquina va a utilizar la red del host, es decir en modo NAT, obtendrá la ip 10.0.2.15. Por otra parte en modo adaptador puente, utilizará una ip disponible de la red actual, existen otras opciones pero estas són las mas relevantes. También podemos añadir más adaptadores de red a la máquina virtual.
